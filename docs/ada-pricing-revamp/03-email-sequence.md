嗨，

付款成功，謝謝你。

1. 進入 Google 商家 → 使用者
2. 邀請 `gbp@adlo.tw`（管理員身分）

## 技術串接備注（→ Kael MailerLite 執行）

- 觸發來源：Stripe Webhook → n8n → MailerLite API
- E02 需要動態插入：報告 PDF 連結（7 天有效期）、診斷發現數量（N）
- E05–E07 需要動態計算折抵到期日期（delivery_date + 30 天）
- E04 `refund_completed` 觸發條件：Stripe refund.updated 狀態 = succeeded
- E03 `refund_requested` 觸發條件：Stripe refund.created 事件
