## 1. 戰略大轉向：從「行銷公司」→「中小店家的日常工具箱」

**舊定位：** adlo 是一家台灣在地行銷代操公司，提供 SEO / 廣告 / GBP 服務
**新定位：** adlo 是一個**每週會讓你打開一次**的店家行銷自助工具站，訂閱只是其中一個入口

### 為什麼這個轉向對「來信率 + 商品化」雙贏？

| 舊模型 | 新模型 |
|--------|--------|
| 客戶看官網 → 評估服務 → 填表 → 等回覆 → 報價 → 簽約 | 客戶來用免費工具 → 分數不及格 → 看到報告 → 留 email → 回頭再用 → 自然相信 → 主動諮詢 |
| 漏斗長、流失率高 | 漏斗短、**每個工具都是獨立漏斗入口** |
| SEO 只能靠文章 | **工具頁本身就是 SEO 長尾殺手**（「Google 商家 健診工具」「關鍵字難度 台灣」）|
| 客戶不會主動回訪 | **工具會被加書籤、每週回來看分數** |

### 每個工具共通設計要素（Dior 規範）

1. **進入頁面 < 3 秒就能操作**（不要 onboarding，不要 wizard）

3. **結果畫面含分享卡**（@vercel/og 自動生成，IG 方形 + Stories 直式）
4. **每次操作都可能觸發 email gate**（Layer 0 → L1 轉換）
5. **頁面底部永遠有 3 個推薦：相關工具 / 相關 SEO 文章 / 深度診斷預約**

## 3. 為什麼這樣能「讓客戶沈迷」？（Hooked 四步驗證）

| Hooked 步驟 | adlo 工具箱如何實踐 |
|------------|---------------------|
| **Trigger（觸發）** | 外部：Threads/IG/Google 搜尋 → 看到工具頁<br>內部：店家老闆每週焦慮「我的排名有沒有進步」 |
| **Action（行動）** | 輸入店名 → 3 秒出分數，幾乎零摩擦 |
| **Variable Reward（變動獎勵）** | 每次分數都可能不同（因為競爭對手也在變），每次都想再查一次 |
| **Investment（投入）** | 留 email → 每週收到「你的分數變化」週報 → 越看越想進步 → 越依賴 adlo |

**關鍵：變動獎勵 ＋ 週期性比較，是老虎機等級的成癮機制。**
店家老闆本來就每天焦慮同行，adlo 等於給他一個合法、有建設性的焦慮出口。

## 4. 商品化收斂：60 天後的付費產品線

免費工具是漏斗頂，付費產品是底。Layer 對應：

```
🟢 Layer 0  /tools/*            免費工具群（7 個，每週一支新工具）
              │
              └→ 使用者留 email、保存歷史、對網站有感情
              │
🟡 Layer 1  /diagnostic         深度診斷報告（現期免費預約，60 天後 NT$1,990）
              │
              └→ 專人報告、建立信任、升級 gate
              │
🔵 Layer 2  /subscribe          三個產品化月訂閱（60 天後上線）
              │
              ├── adlo GBP Auto          NT$3,980/月
              ├── adlo Local SEO Pack    NT$9,800/月  ← 主推
              └── adlo Ads Managed       NT$5,000 + 15%
              │
🟣 Layer 3  Dashboard           會員中心（紀錄工具使用歷史、對比趨勢、週報設定）
```

**商品化關鍵突破：**
付費產品不賣「月工時」，賣「你已經用了三個月免費工具知道自己分數停在 65 分——我們幫你打到 85 分」。
前面免費工具使用歷程＝最好的銷售材料。
