---

## 5. 結果頁規格（7 張卡）

每張卡內容（從上到下）：
1. **星期 + 主題分類 Badge**（週一 · 節慶 / 週二 · 教育 / …）
2. **短標**（≤ 16 字，老闆瞄一眼就知道在講什麼）
3. **正文**（150–250 字，繁中，台灣語感）
4. **建議貼文時間**（09:00 / 12:30 / 18:00）
5. **建議圖片提示**（一句話 = 拍什麼）
6. **動作按鈕**：複製文案 / 換一篇

### 七天主題輪盤（固定順序）

| 星期 | 主題 | 用意 |
|------|------|------|
| 一 | 節慶/時事 | 抓週初聲量 |
| 二 | 教育 | 建立專業形象 |
| 三 | 客戶見證 | 社會證明 |
| 四 | 幕後 | 拉近距離 |
| 五 | 新品/限定 | 拉週末客 |
| 六 | 促銷 | 假日直球 |
| 日 | QA | 收尾 + 解 reservation 焦慮 |

---

## 6. 品牌語感鐵律（生成內容必須遵守）

✅ **要有：**
- 台灣口語（「揪一下朋友」「順手帶一杯」）
- 直球但不過度推銷（「想試試看？這週末來」）
- 在地細節（地名、捷運站、季節）

❌ **絕對禁止：**
- 簡中語感：賦能、打造、優質、極致、傾力、煥新、深度賦能
- 套話：歡迎大家來詢問～、希望對大家有幫助、一起來看看
- 連續驚嘆號：！！！
- emoji 過多（每篇 ≤ 2 個）
- 「業界領先 / 第一 / 最強」

## 7. AI Prompt 範本（給 Kael 之後串 API 用）

```
你是台灣中小店家的社群文案助手，專門寫 Google 商家貼文。
背景：這個店家叫 {{storeName}}，產業是 {{industry}}，本週主打 {{weekTheme}}（若空白就忽略）。

請幫他寫**一週 7 天的 Google 商家貼文初稿**，每天一篇，依照以下規則：

【七天主題分配】（嚴格遵守順序）
週一：節慶或時事連結
週二：產業教育（一個小知識，不能太長）
週三：客戶見證（虛構一則 5 星評論的回應）
週四：幕後故事（現場、人、製程）
週五：新品或限定品介紹
週六：促銷或週末活動
週日：QA（常被問到的一題）

【每篇格式】（JSON）
{
  "day": "週X",
  "category": "節慶|教育|客戶見證|幕後|新品|促銷|QA",
  "title": "≤16 字的短標",
  "content": "150-250 字繁體中文，台灣口語，直白但不推銷",
  "bestTime": "建議貼文時間，HH:MM",
  "imageHint": "建議拍什麼，一句話"
}

【硬性禁止】
- 簡中詞彙：賦能、打造、優質、極致、傾力、煥新、深度
- 套話：歡迎大家來詢問、希望對大家有幫助、今天要跟大家分享
- 連續驚嘆號（！！！）
- 每篇 emoji 不超過 2 個

【口吻範例（要學的）】
「上週四下午有個媽媽帶兩個小孩來，說小朋友吃完整盤蛋餅還想再點一份。我們其實沒做什麼特別的，就是每天 4:30 揉麵糰。皮要 Q，得從深夜開始。」

回傳一個包含 7 個物件的 JSON array，不要任何前後說明文字。
```
