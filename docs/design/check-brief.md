## 👤 目標受眾

**核心輪廓：**
- 35-55 歲中小企業主 / 店家老闆
- 有 Google 商家檔案（GBP）但不懂優化
- 會看 LINE、偶爾滑 IG，很少用 Dcard / PTT
- 對「分數」有感，對「SEO 指標」無感

**使用情境：**
1. 朋友在 LINE 分享「我 X 分，你呢？」→ 好奇點進來
2. 看到同業在 FB 社團貼「我 X 分」→ 自己想測
3. 被動搜尋「Google 商家評分怎麼看」→ 搜到 /check

**心理狀態：**
- 既想知道自己店家表現，又怕被看到分數低
- 分數高 → 炫耀傾向強（截圖分享）
- 分數低 → 求救傾向強（問「怎麼辦」）
- 分數中 → 好奇傾向強（「怎麼變高」）

### 互動主軸

```
1. 首屏輸入框（單一 CTA，超大）
   - placeholder: 「貼上 Google 地圖連結，或輸入店名 + 地區」
   - 按鈕：「免費查分數」
2. 載入動畫（3-5 秒，抓資料 + 評分感）
3. 分數展示頁（0-100 分環狀進度條 + 6 項指標）
4. 分享卡產生區（方形 + 直式兩版預覽）
5. 升級 CTA（診斷包 NT$1,990 —「想知道為什麼嗎？」）
```

### Rate Limit UI 處理

```
第 1-3 次：正常流程
第 4 次起：
  - 卡片彈出：「已查 3 次，留下 email 繼續使用」
  - 單欄 email 表單（無 spam 感）
  - 按鈕：「解鎖 10 次查詢」（不是「訂閱電子報」）
  - 信任文案：「只用於通知健檢優化建議，隨時退訂」
```

### FAQ（6 題）

1. **查分數會收到 adlo 的業配電話嗎？**
   不會。我們連電話欄位都沒有。你留 email 只會收到健檢優化建議，想退訂就退。

2. **我的店資料會被存嗎？**
   只存你查詢的 URL 和分數結果，24 小時後自動刪。不抓客人名單、不抓聯絡人。

3. **分數怎麼算的？**
   6 項公開指標加權：商家完整度 20% / 評論數 20% / 回覆率 15% / 照片 15% / 關鍵字 15% / 在地競爭 15%。都是 Google 公開的資料。

4. **為什麼第 4 次要 email？**
   一台電腦查 3 次通常夠，再下去多半是幫別人查。留 email 是為了確保你是真的在乎這件事，不是爬蟲。

5. **分數低是不是代表店很爛？**
   不是。低分只代表「Google 呈現給客人的樣子沒補齊」，跟實際服務品質無關。我們就是幫你把這一塊補起來。

6. **免費查完還要付錢嗎？**
   查分數永遠免費。如果你想知道「為什麼是這分數、怎麼修」，才需要 NT$1,990 的完整報告。

## 🇹🇼 台灣語感確認（Ada 已過）

✅ 通過以下檢查：
- 使用「店家」而非「商戶」/「商家」過度（簡中感）
- 使用「客人」而非「用戶」/「顧客」（口語感）
- 使用「幾分」而非「多少分」（台灣語序）
- 使用「拿到」而非「獲得」（生活感）
- 使用「被看見」而非「可見性」（翻譯腔）
